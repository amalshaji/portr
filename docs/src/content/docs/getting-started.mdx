Welcome to **Portr** - the self-hosted tunnel solution that gives you complete control over your development infrastructure!

<Aside type="tip">
  **New to tunneling?** Portr allows you to expose local services (like your development server) to the internet through secure tunnels, making them accessible from anywhere.
</Aside>

## What is Portr?

Portr is a **self-hosted tunnel solution** designed for local development who need:

- 🔒 **Complete data control** - Your tunnels run on your infrastructure
- 👥 **User management** - Built-in user management and access controls
- 📊 **Advanced monitoring** - Real-time connection monitoring and request inspection
- 🚀 **Development features** - Custom domains, SSL certificates, and more

## Getting Started

<Steps>

1. **Deploy the Portr Server**

   Set up your Portr server on your preferred infrastructure:

   ```bash
   # Using Docker (recommended)
   docker run -p 8080:8080 -p 8081:8081 portr/server
   ```

   → [Complete server setup guide](/server/)

2. **Install the Portr Client**

   Download and install the client for your platform:

   ```bash
   # macOS/Linux
   curl -L https://github.com/amalshaji/portr/releases/latest/download/portr-cli | bash

   # Windows
   # Download from GitHub releases
   ```

   → [Client installation guide](/client/installation/)

3. **Create Your First Tunnel**

   Start tunneling in seconds:

   ```bash
   # HTTP tunnel
   portr http 3000

   # TCP tunnel
   portr tcp 5432

   # WebSocket tunnel
   portr ws 8080
   ```

   → [Tunnel configuration guide](/client/http-tunnel/)

</Steps>
