## Why Portr?

Portr is a **self-hosted tunnel solution** designed for teams who need secure, reliable access to local development environments. Unlike other tunnel services, Portr gives you complete control over your infrastructure while providing enterprise-grade features.

<CardGrid stagger>
  <Card title="🚀 Simple & Fast" icon="rocket">
    Get started in seconds. Expose any local service with a single command:

    ```bash
    portr http 3000
    ```
  </Card>

  <Card title="🔒 Self-Hosted Security" icon="shield">
    Keep your data secure with full control over your tunnel infrastructure. No third-party dependencies.
  </Card>

  <Card title="👥 Team-First Design" icon="group">
    Built for collaboration with team management, role-based access control, and shared tunnel configurations.
  </Card>

  <Card title="📊 Advanced Monitoring" icon="chart">
    Real-time connection monitoring, request inspection, and comprehensive logging for debugging.
  </Card>
</CardGrid>

## Supported Protocols

<CardGrid>
  <Card title="HTTP/HTTPS Tunnels" icon="globe">
    Expose web applications with automatic HTTPS certificates and custom domains.

    ```bash
    portr http 8080 --subdomain myapp
    ```
  </Card>

  <Card title="TCP Tunnels" icon="network">
    Tunnel any TCP service like databases, SSH servers, or custom protocols.

    ```bash
    portr tcp 5432 --port 8432
    ```
  </Card>

  <Card title="WebSocket Tunnels" icon="lightning">
    Real-time WebSocket connections with full bidirectional communication.

    ```bash
    portr ws 9001 --subdomain socket
    ```
  </Card>
</CardGrid>
